What is Wireframing? Wireframing is a process where designers draw overviews of interactive products to establish the structure and flow of possible design solutions.These outlines reflect user and business needs. Wireframes are basic blueprints that help teams align on requirements, keeping UX design conversations focused and constructive

Projects

My placement

Every year, campus placement produces a large number of recruits for every company. Through this placement, many students are getting jobs with good packages. In order to help companies to find the best candidates and allow students to be hired by the best companies, we are introducing the "myPlacement" application. By removing the need for an intermediary and allowing the student and company to communicate directly after verifying each other, this application will lessen the workload of the TPO (Training and Placement Officer).

E-Ration

The E-Ration is the web portal that provides management of ration in online mode. E-Ration provides a multi-mode payment system, an email notifying service for users to notify that ration is available for user. As E-Ration is a web portal which means it has to be faster and more convenient for every user while they are interacting with it. The E-Ration is made up of many modules such as admin module, user module. We can split the modules and with the help of this model, we can work and split the modules differently using this model. In an incremental model, the feedback of the client or the changes given by the client can be implemented in the next level. Implementing the changes will not create problems in the incremental model. The concept of E-Ration includes frequent changes in the inputs. That's why we had chosen the incremental model for the E-ration.
